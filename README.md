# Mi primera web en Django - Clínica

Esta pagina, simula una clínica donde se pueden cargar pacientes, médicos y turnos, y también buscar pacientes.

## ¿Que tiene?

- Herencia de plantillas
- 4 clases en models: Paciente, Medico, Turno, Buscar turno.
- Formularios para cargar datos de cada uno, incluyendo el tiempo y dia.
- Un formulario para buscar pacientes en la base de datos.

## Cómo usarlo

Ir a "Nuevo Paciente", "Nuevo Médico" o "Nuevo Turno" para cargar datos.

Ir a "Buscar" y escribir el nombre de un paciente para buscarlo.
